# twitterProfileTracker
> Simple tool for logging tweets of a profile to mysql database

- Kurulum icin:
1. ` npm install`
2. `settings.conf` icerigini kendi bilgilerinizle guncelleyin.

------------

- Calistirmak icin:
```shell
npm run start
```
